# 场景教程

快速上手 EDGE 智能边缘平台的核心功能，每个教程独立完整，按需阅读。

## 教程列表

| 教程 | 场景 | 时间 |
|------|------|------|
| [教程1：节点接入](./node-access) | 将边缘 Linux 机器接入平台 | ~5 分钟 |
| [教程2：应用部署](./app-deployment) | 镜像接入 + 应用创建 + 部署 | ~10 分钟 |
| [教程3：应用上架与升降级](./app-publish-upgrade) | 上架到商店 + 多租户复用 + 版本切换 | ~15 分钟 |
| [教程4：监控展示](./monitoring) | 查看集群、节点、工作负载监控 | ~5 分钟 |
| [教程5：创建租户与启用 EdgeX](./tenant-edgex) | 新建租户空间 + 开启 IoT 服务 | ~10 分钟 |

## 推荐路径

**初次使用**：教程1（节点接入）→ 教程2（应用部署）→ 教程4（监控展示）

**IoT 场景**：教程5（创建租户）→ 教程1（节点接入）→ 查看设备管理

**应用发布**：教程2（应用部署）→ 教程3（上架与升降级）
