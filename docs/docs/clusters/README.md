# 集群管理概述

> **导航路径**: 顶部导航栏 > 集群
> **访问地址**: `/boss/clusters`
> **所需权限**: 平台管理员或集群管理员

## 功能说明

集群管理提供对平台中所有集群的统一管理。您可以查看集群列表、进入集群详情查看资源使用情况、管理节点组和项目、配置集群成员权限。

## 集群侧边栏

进入集群详情后，左侧边栏包含以下菜单：

| 菜单 | 说明 |
|------|------|
| 概览 | 集群资源统计、实时用量、配额、告警和排行 |
| 节点 | 集群中的云端节点和边缘节点 |
| 节点组 | 节点的逻辑分组管理 |
| 算力管理 | GPU 显卡规格和算力配比 |
| 项目 | 集群中的项目（命名空间）管理 |
| 应用负载 | 工作负载、容器组、服务等 |
| 配置 | 配置字典和保密字典 |
| 存储 | 存储卷管理 |
| 监控告警 | 集群指标、告警和趋势图 |
| 集群设置 | 基本信息、集群成员、集群角色 |
