# 设备模版

> **导航路径**: 智能边缘平台 > 选择工作台 > 选择节点组 > 边缘资源 > 设备模版
> **访问地址**: `/tenant/{租户空间ID}/nodegroups/{节点组ID}/device-profile`
> **所需权限**: 租户空间管理员或成员

## 功能说明

设备模版（配置文件）定义了物联网设备的数据模型，包括设备属性、命令和通信参数。创建设备时需要关联一个设备模版，设备按照模版定义的格式上报数据和接收指令。

## 页面概览

![设备模版列表](./images/iot-device-profile-list.png)

设备模版列表包含以下信息：

| 列 | 说明 |
|------|------|
| 配置文件名称 | 模版的名称和描述 |
| 设备型号 | 适用的设备型号 |
| 厂商 | 设备厂商名称 |
| 协议 | 设备通信协议 |
| 属性数量 | 模版定义的属性数量 |
| 命令数量 | 模版定义的命令数量 |
| Namespace | 模版所在的项目 |
| 创建时间 | 模版的创建时间 |
| 时长 | 模版创建至今的时间 |

## 操作指南

### 操作一：查看设备模版列表

**操作步骤**

1. 在工作台侧边栏中，展开 **边缘资源**

2. 点击 **设备模版**

3. 查看所有模版的名称、厂商和属性/命令数量

4. 使用搜索框按名称查找模版

**操作结果**

查看到节点组内所有设备模版及其配置信息。

### 操作二：创建设备模版

**操作步骤**

1. 在模版列表页面，点击 **创建配置文件** 按钮

2. 填写模版基本信息（名称、描述、厂商、设备型号）

3. 定义设备属性（属性名称、数据类型、读写权限）

4. 定义设备命令（命令名称、参数）

5. 点击 **确认** 创建模版

**操作结果**

设备模版创建成功，可在创建设备时选择使用。

### 操作三：查看模版详情

**操作步骤**

1. 在模版列表中，点击目标模版的名称

2. 进入模版详情页面，查看属性列表和命令列表

**操作结果**

了解模版定义的完整属性和命令配置。

## 常见问题

### 问题：创建设备时找不到模版
**现象**：创建设备时配置文件下拉列表为空
**原因**：当前节点组内未创建任何设备模版
**解决**：先创建设备模版，再创建设备

### 问题：模版属性数量与实际不符
**现象**：模版显示的属性数量与预期不一致
**原因**：部分属性可能在创建后被修改或删除
**解决**：进入模版详情查看完整的属性列表
