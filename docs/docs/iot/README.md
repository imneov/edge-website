# IoT 设备管理概述

> **导航路径**: 智能边缘平台 > 选择工作台 > 边缘资源
> **所需权限**: 租户空间管理员或成员

## 功能说明

IoT 设备管理用于管理接入边缘节点的物联网设备。您可以在工作台的边缘资源中查看设备列表、设备状态，以及管理设备配置文件（设备模版）。设备管理功能位于节点组维度下，每个节点组独立管理其设备。

## 入口位置

IoT 设备管理位于工作台侧边栏的 **边缘资源** 下，包含：

| 菜单 | 说明 |
|------|------|
| 设备管理 | 查看和管理节点组内的物联网设备 |
| 设备模版 | 管理设备配置文件，定义设备属性和命令 |
