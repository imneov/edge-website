# OTA 管理概述

> **导航路径**: 控制台 > OTA
> **访问地址**: `/boss/ota`
> **所需权限**: 平台管理员

## 功能说明

OTA（Over-the-Air）模块用于远程管理边缘节点，支持节点注册、状态监控、远程命令执行、文件读写和批量任务管理。通过 OTA 模块，管理员可以对大规模边缘节点进行统一运维。

## OTA 功能

OTA 模块包含两个主要功能：

| 功能 | 说明 |
|------|------|
| 节点接入 | 注册和管理边缘节点，查看节点状态，远程执行命令和文件管理 |
| 任务管理 | 创建批量任务，对多个节点同时执行命令、读写文件或运行 Playbook |
