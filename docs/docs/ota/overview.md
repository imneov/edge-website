# OTA 简介

> **导航路径**: 控制台 > OTA
> **所需权限**: 平台管理员

## 功能说明

OTA 模块提供边缘节点的远程管理能力。当边缘节点部署在远程位置、无法现场操作时，OTA 模块可以帮助管理员远程注册节点、监控状态、执行命令和管理文件。

## 核心功能

### 节点接入

- 支持单个添加和批量导入边缘节点
- 实时查看节点的在线状态和系统信息
- 远程执行命令（类似 SSH）
- 远程浏览和管理节点文件
- 查看节点已安装的软件包

### 任务管理

- 创建批量任务，同时对多个节点执行操作
- 支持四种任务类型：执行命令、读取文件、写入文件、运行 Playbook
- 查看任务执行进度和每个节点的执行结果
- 任务历史记录查询

## 使用场景

| 场景 | 使用方式 |
|------|----------|
| 检查节点状态 | 节点接入 > 查看节点列表和在线状态 |
| 远程排查问题 | 节点接入 > 选择节点 > 执行命令 |
| 批量更新配置 | 任务管理 > 创建写入文件任务 |
| 批量安装软件 | 任务管理 > 创建执行命令任务 |
| 自动化运维 | 任务管理 > 创建 Playbook 任务 |
