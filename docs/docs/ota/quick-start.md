# 快速上手

> **所需权限**: 平台管理员

## 前提条件

- 已部署边缘智能管理平台
- 具有平台管理员权限
- 边缘节点可通过网络访问

## 步骤一：注册第一个节点

1. 进入 **控制台 > OTA > 节点接入**

2. 点击 **添加节点** > **单个添加**

3. 填写节点的 IP 地址、端口和认证信息

4. 点击 **确认**

5. 等待节点状态变为「在线」

## 步骤二：验证节点连接

1. 在节点列表中，点击刚添加的节点名称

2. 进入 **执行命令** 标签页

3. 输入测试命令：`hostname`

4. 点击 **执行**

5. 确认返回了节点的主机名

## 步骤三：创建第一个任务

1. 进入 **控制台 > OTA > 任务管理**

2. 点击 **创建任务**

3. 选择任务类型 **执行命令**

4. 输入命令：`uptime`

5. 选择目标节点

6. 点击 **确认**

7. 在任务列表中查看执行结果
