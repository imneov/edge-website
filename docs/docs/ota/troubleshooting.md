# 常见问题

## 节点接入问题

### 问题：节点添加后一直离线
**现象**：添加节点后，状态一直显示「离线」
**原因**：
- 节点 IP 地址或端口填写错误
- 节点网络不可达（防火墙、网络隔离）
- 节点上的 Agent 未安装或未启动
**解决**：
1. 确认节点 IP 和端口是否正确
2. 从平台所在服务器尝试连接节点（确认网络通畅）
3. 登录节点确认 Agent 服务已启动

### 问题：节点频繁掉线
**现象**：节点状态在「在线」和「离线」之间频繁切换
**原因**：网络不稳定，或节点负载过高导致心跳超时
**解决**：
1. 检查节点网络连接质量
2. 检查节点 CPU 和内存使用率
3. 确认 Agent 进程未被其他进程干扰

## 任务执行问题

### 问题：任务执行超时
**现象**：任务长时间处于「进行中」状态
**原因**：命令执行时间超过超时限制，或节点响应缓慢
**解决**：
1. 检查目标节点是否在线
2. 尝试在节点详情中手动执行相同命令
3. 对于耗时较长的操作，考虑拆分为多个小任务

### 问题：写入文件任务失败
**现象**：写入文件任务显示执行失败
**原因**：目标路径不存在、磁盘空间不足或权限不足
**解决**：
1. 确认目标目录在节点上存在
2. 检查节点磁盘空间
3. 确认 Agent 有写入目标路径的权限

### 问题：Playbook 任务部分节点失败
**现象**：同一个 Playbook 在部分节点成功，部分节点失败
**原因**：节点环境不一致（缺少依赖、目录结构不同）
**解决**：
1. 在任务详情中查看失败节点的具体错误信息
2. 对比成功和失败节点的环境差异
3. 在 Playbook 中添加环境检查步骤
