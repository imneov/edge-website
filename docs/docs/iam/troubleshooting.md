# 常见问题

## 登录问题

### 问题：用户无法登录
**现象**：输入正确的用户名和密码后，提示登录失败
**原因**：
- 用户账户被禁用
- 密码输入错误（注意大小写）
- 用户名不存在
**解决**：
1. 确认用户名拼写正确
2. 确认密码正确（包括大小写和特殊字符）
3. 联系平台管理员检查账户状态是否为「活跃」
4. 如忘记密码，联系管理员重置密码

### 问题：登录后页面空白
**现象**：用户登录成功后看不到任何资源
**原因**：用户只有 platform-regular 角色，尚未被邀请加入任何租户空间
**解决**：由平台管理员将用户邀请到至少一个租户空间中

## 权限问题

### 问题：用户看不到某个菜单
**现象**：用户登录后，侧边栏缺少某些菜单项
**原因**：用户当前的角色没有对应功能的访问权限
**解决**：
1. 在用户详情的「用户角色」标签页中检查角色分配
2. 根据用户需要的功能，分配对应的角色

### 问题：操作时提示权限不足
**现象**：用户执行操作（如创建、删除、编辑）时收到权限拒绝的提示
**原因**：用户的角色只有查看权限，没有操作权限
**解决**：为用户分配包含对应操作权限的角色

### 问题：用户无法访问特定租户空间
**现象**：用户在工作台看不到某个租户空间
**原因**：用户未被邀请到该租户空间
**解决**：由租户空间管理员在成员管理中邀请该用户

## 角色问题

### 问题：分配角色后权限未生效
**现象**：为用户分配角色后，用户仍然无法执行对应操作
**原因**：浏览器缓存了旧的权限信息
**解决**：
1. 让用户退出登录后重新登录
2. 清除浏览器缓存后重试

### 问题：无法删除系统角色
**现象**：尝试删除 platform-admin 等系统角色时失败
**原因**：系统角色是平台内置的，不支持删除或修改
**解决**：系统角色无法删除，如不需要可以取消对用户的分配
