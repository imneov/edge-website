# 访问控制

访问控制模块用于管理平台的用户账户和角色权限。通过用户管理和角色管理，控制不同用户对平台资源的访问权限。

## 侧边栏菜单

> **导航路径**: 控制台 > 访问控制
> **访问地址**: `/boss/access/users`

| 菜单项 | 说明 | 文档链接 |
|--------|------|----------|
| 用户管理 | 创建和管理用户账户 | [用户管理](./user-management.md) |
| 角色管理 | 查看和管理角色及其权限 | [角色管理](./role-management.md) |
