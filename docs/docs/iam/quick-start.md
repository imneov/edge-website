# 快速上手

> **所需权限**: 平台管理员

## 前提条件

- 已部署边缘智能管理平台
- 具有平台管理员权限

## 步骤一：创建用户

1. 进入 **控制台 > 访问控制 > 用户管理**

2. 点击 **新建用户**

3. 填写用户信息：
   - 用户名（如 `developer-1`）
   - 邮箱（如 `dev1@example.com`）
   - 密码

4. 点击 **确定**

## 步骤二：为用户分配平台角色

1. 进入 **控制台 > 访问控制 > 角色管理**

2. 点击 `platform-regular` 角色名称进入详情

3. 点击 **授权用户** 标签页

4. 点击 **分配用户**，选择刚创建的用户

5. 点击 **确认**

## 步骤三：邀请用户到租户空间

1. 进入 **租户空间** > 选择目标租户空间

2. 进入 **租户空间设置 > 租户空间成员**

3. 点击 **邀请成员**

4. 搜索并选择刚创建的用户

5. 分配租户空间角色

6. 点击 **确认**

## 步骤四：验证用户访问

1. 使用新创建的用户名和密码登录平台

2. 确认用户可以在工作台中看到被分配的租户空间

3. 确认用户可以访问租户空间内的资源
