# 租户空间概述

> **导航路径**: 顶部导航栏 > 租户空间
> **访问地址**: `/boss/workspaces`
> **所需权限**: 平台管理员

## 功能说明

租户空间是平台中的资源隔离单位。每个租户空间拥有独立的项目、应用模板和成员权限。平台管理员在此创建和管理租户空间，租户空间管理员在各自的租户空间内管理项目和成员。

## 租户空间侧边栏

进入租户空间详情后，左侧边栏包含以下菜单：

| 菜单 | 子菜单 | 说明 |
|------|--------|------|
| 概览 | — | 物理资源、应用资源用量、基本信息、资源排行 |
| 项目管理 | — | 租户空间内的项目列表和管理 |
| 应用管理 | 应用模板 | 租户空间内的应用模板 |
| 租户空间设置 | 基本信息 | 查看和编辑租户空间信息 |
| | 租户空间成员 | 成员邀请和角色分配 |
| | 租户空间角色 | 租户空间级别的角色定义 |
