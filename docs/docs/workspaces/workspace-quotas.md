# 资源配额

> **导航路径**: 租户空间 > 选择租户空间 > 概览 > 物理资源
> **所需权限**: 平台管理员或租户空间管理员

## 功能说明

资源配额通过租户空间概览页面的物理资源区域展示。您可以在此查看 CPU、内存和持久卷的使用量与配额限制，了解资源分配情况。

## 配额信息

租户空间概览页面的物理资源区域显示以下配额：

| 资源 | 说明 |
|------|------|
| CPU | CPU 已使用量 / 配额上限 |
| 内存 | 内存已使用量 / 配额上限 |
| 持久卷 | 持久卷已使用量 / 配额上限 |

每个资源以仪表盘形式展示使用比例，方便快速了解资源消耗情况。

## 操作指南

### 操作一：查看资源配额

**操作步骤**

1. 进入目标租户空间，点击侧边栏的 **概览**

2. 在物理资源区域查看 CPU、内存和持久卷的用量和配额

**操作结果**

了解租户空间的资源使用率和剩余可用资源。

## 常见问题

### 问题：物理资源显示为 0
**现象**：概览页面的物理资源区域显示用量为 0
**原因**：租户空间内没有运行中的工作负载，或监控数据未就绪
**解决**：确认租户空间内有项目和运行中的工作负载

### 问题：资源使用量超过配额
**现象**：某项资源的使用率接近或超过 100%
**原因**：租户空间内的工作负载消耗了大量资源
**解决**：联系平台管理员调整配额，或清理不需要的工作负载释放资源
