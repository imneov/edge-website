# 常见问题

## 镜像源问题

### 问题：镜像源连接失败
**现象**：测试连接后，状态显示「连接失败」
**原因**：
- 镜像仓库地址填写错误（多了 `https://` 前缀或路径后缀）
- 用户名或密码错误
- 平台服务器无法访问镜像仓库（网络不通、防火墙阻止）
- 镜像仓库服务已停止
**解决**：
1. 确认地址格式正确，只填写域名（如 `harbor.example.com`）
2. 确认用户名和密码正确
3. 从平台所在服务器测试是否能访问镜像仓库
4. 联系镜像仓库管理员确认服务是否正常

### 问题：需要更新密码但无法修改
**现象**：镜像仓库密码已更改，需要更新平台上的凭证
**原因**：平台不支持直接修改镜像源的核心配置
**解决**：删除现有的镜像源，使用新密码重新创建

## 镜像同步问题

### 问题：镜像同步失败
**现象**：镜像的同步状态显示「同步失败」
**原因**：
- 镜像源连接异常（凭证失效或网络中断）
- 镜像路径在仓库中不存在
- 镜像源账户没有访问该镜像的权限
**解决**：
1. 先检查对应镜像源的连接状态是否正常
2. 确认镜像路径在镜像仓库中确实存在
3. 确认镜像源账户有权限访问该镜像
4. 删除后重新添加

### 问题：镜像长时间处于「同步中」
**现象**：镜像添加后长时间无法完成同步
**原因**：网络连接不稳定，或镜像文件较大
**解决**：
1. 检查平台到镜像仓库的网络连接是否正常
2. 等待一段时间后刷新页面
3. 如持续无变化，删除后重新添加

### 问题：部分镜像同步成功，部分失败
**现象**：同一个镜像源下的镜像，有的同步成功有的失败
**原因**：失败的镜像可能路径错误，或镜像源账户对不同镜像的权限不同
**解决**：
1. 检查失败镜像的路径是否正确
2. 确认镜像源账户对失败镜像有读取权限
