# 快速上手

> **所需权限**: 平台管理员

## 前提条件

- 已部署边缘智能管理平台
- 具有平台管理员权限
- 有一个可访问的镜像仓库（如 Harbor）及其登录凭证

## 步骤一：添加镜像源

1. 进入 **控制台 > 镜像仓库 > 镜像源管理**

2. 点击 **添加镜像源**

3. 填写镜像仓库的连接信息：
   - 选择命名空间
   - 输入镜像仓库地址（如 `harbor.example.com`）
   - 输入用户名和密码

4. 点击 **创建**

5. 在镜像源列表中，点击操作菜单 > **测试连接**，确认状态为「连接成功」

## 步骤二：添加镜像

1. 切换到 **镜像管理** 页面

2. 点击 **添加镜像**

3. 选择命名空间和刚添加的镜像源

4. 浏览并选择要同步的镜像

5. 确认添加

## 步骤三：验证同步

1. 在镜像仓库列表中查看新添加的镜像

2. 等待同步状态变为「已同步」

3. 同步完成后，该镜像即可在应用部署中使用
