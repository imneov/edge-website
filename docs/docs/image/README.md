# 镜像管理

镜像管理模块用于管理容器镜像的来源和同步。通过配置镜像源（仓库凭证）和镜像仓库（具体镜像），确保边缘节点能够正常拉取所需的容器镜像。

## 侧边栏菜单

> **导航路径**: 控制台 > 镜像仓库
> **访问地址**: `/boss/images/registry`

| 菜单项 | 说明 | 文档链接 |
|--------|------|----------|
| 镜像源管理 | 管理镜像仓库的访问凭证和连接 | [镜像源管理](./registry-management.md) |
| 镜像管理 | 管理需要同步的具体镜像 | [镜像仓库管理](./repository-management.md) |

## 其他入口

镜像管理功能还可以在以下位置访问：

| 入口 | 路径 | 说明 |
|------|------|------|
| 租户空间项目 | 租户空间 > 项目 > 镜像源/镜像仓库 | 项目级别的镜像管理 |
| 工作台项目 | 工作台 > 项目 > 镜像源/镜像仓库 | 工作台内项目级别的镜像管理 |
