# 配置管理

> **导航路径**: 智能边缘平台 > 工作台 > 配置管理
> **所需权限**: 工作台管理员或资源管理权限

## 功能说明

配置管理允许您管理工作台内应用使用的配置数据和敏感信息。配置字典用于存放非敏感的配置数据，保密字典用于存放密码、密钥等敏感信息。

## 操作指南

### 操作一：查看配置字典列表

**操作步骤**

1. 在工作台侧边栏中，展开 **配置管理**，点击 **配置字典**

2. 配置字典列表显示当前项目中所有配置项的名称、数据条数和创建时间

3. 使用页面顶部的搜索框按名称搜索

**操作结果**

查看到当前项目中的所有配置字典。

### 操作二：查看保密字典列表

**操作步骤**

1. 在工作台侧边栏中，展开 **配置管理**，点击 **保密字典**

2. 保密字典列表显示名称、类型、数据条数和创建时间

**操作结果**

查看到当前项目中的所有保密字典。保密字典的值默认不显示明文。

### 操作三：查看配置详情

**操作步骤**

1. 在配置字典或保密字典列表中，点击目标配置项的名称

2. 在详情页面查看配置的键值对内容

**操作结果**

查看到该配置项的所有键值对数据。

## 常见问题

### 问题：配置字典列表为空
**现象**：进入配置字典页面后没有任何数据
**原因**：当前项目中尚未创建配置字典，或系统配置被隐藏
**解决**：如需为应用添加配置，请在部署工作负载时创建配置字典

### 问题：保密字典内容无法查看
**现象**：保密字典的值显示为密文
**原因**：保密字典为了安全默认不显示明文
**解决**：点击「显示」按钮查看明文内容（需要相应权限）
