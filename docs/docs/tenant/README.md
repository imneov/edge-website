# 工作台概述

> **导航路径**: 顶部导航栏 > 智能边缘平台
> **访问地址**: `/tenant`
> **所需权限**: 平台用户（需被邀请至至少一个工作台）

## 功能说明

工作台是边缘智能管理平台的核心工作区域，每个工作台对应一个独立的边缘计算环境。您可以在工作台中管理边缘节点、部署应用、配置服务，并管理团队成员的访问权限。

## 页面概览

![工作台列表](./images/tenant-workspace-list.png)

进入「智能边缘平台」后，您将看到所有可访问的工作台卡片列表。每张卡片显示工作台的名称、ID、运行状态、所属集群和管理员信息。
