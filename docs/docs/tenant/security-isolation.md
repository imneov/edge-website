# 应用模板

> **导航路径**: 智能边缘平台 > 工作台 > 边缘应用 > 应用模板
> **访问地址**: `/tenant/{工作台ID}/app-templates`
> **所需权限**: 工作台管理员或应用管理权限

## 功能说明

应用模板页面展示当前工作台中可用的应用模板。您可以浏览模板列表、查看模板详情，并基于模板部署应用到项目中。

## 操作指南

### 操作一：浏览应用模板

**操作步骤**

1. 在工作台侧边栏中，展开 **边缘应用**，点击 **应用模板**

2. 查看当前工作台可用的应用模板列表

3. 使用搜索框按模板名称搜索

**操作结果**

查看到当前工作台中所有可用的应用模板。

### 操作二：查看模板详情

**操作步骤**

1. 在应用模板列表中，点击目标模板的名称

2. 在详情页面查看模板的版本列表、配置说明和使用文档

**操作结果**

查看到该应用模板的详细信息和可用版本。

### 操作三：基于模板部署应用

**操作步骤**

1. 在模板详情页面，选择要部署的版本

2. 点击 **部署** 按钮

3. 在部署配置页面，选择目标项目并填写应用配置

4. 点击 **确认** 完成部署

**操作结果**

应用部署成功后，您可以在「工作负载」页面查看运行状态。

## 常见问题

### 问题：看不到任何应用模板
**现象**：应用模板列表为空
**原因**：当前工作台尚未关联应用模板仓库，或模板尚未发布
**解决**：联系平台管理员在租户空间中配置应用模板仓库

### 问题：模板部署失败
**现象**：点击部署后提示错误
**原因**：可能是项目资源配额不足，或镜像拉取失败
**解决**：检查项目的资源配额是否充足；确认边缘节点能否访问镜像仓库
