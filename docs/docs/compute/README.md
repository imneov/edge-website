# 算力管理概述

> **导航路径**: 集群 > 选择集群 > 算力管理
> **所需权限**: 平台管理员或集群管理员

## 功能说明

算力管理用于查看和管理集群中的 GPU 资源。您可以查看所有 GPU 设备的状态和使用情况，创建算力配比模板定义 GPU 切分策略，管理显卡规格定义。

## 算力管理菜单

在集群侧边栏中，算力管理包含以下子菜单：

| 菜单 | 说明 |
|------|------|
| 算力资源 | 查看集群中所有 GPU 设备的状态、分配率和使用率 |
| 算力配比 | 管理 GPU 切分模板，定义虚拟 GPU 的切分数量和超配比 |
| 显卡规格 | 查看和管理 GPU 设备型号，包括物理卡和虚拟卡规格 |

## 其他入口

除了集群级别的算力管理，GPU 资源还可以在以下位置查看：

| 入口 | 路径 | 说明 |
|------|------|------|
| 节点组详情 | 集群 > 节点组 > 选择节点组 > 算力资源 | 查看节点组内的 GPU 设备 |
| 工作台 | 智能边缘平台 > 选择工作台 > 算力资源 | 租户查看可用的 GPU 设备 |
