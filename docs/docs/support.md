# 技术支持

## 联系方式

### 技术支持邮箱

- **邮箱**：support@theriseunion.io
- **工作时间**：周一至周五 9:00 - 18:00（GMT+8）
- **响应时间**：工作日 24 小时内回复

### 问题反馈

如果您在使用平台过程中遇到问题或有改进建议，欢迎通过以下方式反馈：

- **问题报告**：在项目仓库提交 Issue
- **改进建议**：在项目仓库提交 Pull Request
- **技术讨论**：在项目讨论区发起讨论

## 提交问题指南

为了帮助我们更快地定位和解决您的问题，提交问题时请提供以下信息：

### 基本信息

- **平台版本**：当前使用的平台版本号
- **浏览器信息**：浏览器类型和版本
- **操作系统**：客户端操作系统类型

### 问题描述

1. **问题现象**：描述具体的问题表现
2. **复现步骤**：列出导致问题的操作步骤
3. **预期行为**：描述您期望的正确行为
4. **实际行为**：描述实际发生的行为
5. **截图/日志**：提供相关截图或错误日志

### 问题分类

| 优先级 | 说明 | 示例 |
|--------|------|------|
| 紧急 | 平台无法访问或核心功能不可用 | 无法登录、集群失联 |
| 高 | 主要功能异常但有替代方案 | 应用部署失败、权限配置错误 |
| 中 | 非核心功能问题 | 监控数据延迟、UI 显示异常 |
| 低 | 改进建议或文档问题 | 文档错误、UI 优化建议 |

## 自助排查

在联系技术支持前，建议先尝试以下自助排查步骤：

1. **查看文档**：参考 [常见问题](/docs/faq) 中是否有相关解答
2. **检查日志**：查看浏览器控制台和后端日志
3. **验证网络**：确认网络连接正常
4. **清除缓存**：清除浏览器缓存后重试
5. **检查权限**：确认当前用户具有相应权限

## 文档反馈

如果您发现文档中的错误或有改进建议：

- **内容错误**：请提交 Issue 并标注错误位置
- **内容缺失**：请提交 Issue 描述缺失的内容
- **表述不清**：请提交 Issue 说明困惑之处
