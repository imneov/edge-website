# 应用管理概述

> **导航路径**: 顶部导航栏 > 应用商店 / 控制台 > 应用商店管理
> **所需权限**: 所有用户可浏览应用商店，管理员可管理应用

## 功能说明

应用管理包含两个核心功能模块：

- **应用商店**：所有用户可以浏览、搜索和部署应用
- **应用商店管理**：管理员可以管理应用、分类和审核版本发布

## 模块入口

| 模块 | 导航路径 | 功能 |
|------|----------|------|
| 应用商店 | 顶部导航栏 > 应用商店 | 浏览和部署已上架的应用 |
| 应用商店管理 | 控制台 > 应用商店管理 | 管理应用列表、分类和审核 |
| 租户空间应用模板 | 租户空间 > 应用管理 > 应用模板 | 管理租户空间内的应用模板 |
