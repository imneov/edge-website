# 应用实例

> **导航路径**: 工作台 > 选择项目 > 边缘应用 > 应用部署
> **所需权限**: 租户空间管理员或项目成员

## 功能说明

应用实例页面展示已部署到项目中的所有应用。您可以查看每个应用的运行状态、版本信息，以及管理应用的生命周期（升级、删除等）。

## 页面概览

应用实例列表显示以下信息：

| 列 | 说明 |
|------|------|
| 名称 | 应用部署的名称 |
| 状态 | 运行状态（运行中、已停止、异常等） |
| 版本 | 当前部署的应用版本 |
| 更新时间 | 最后更新时间 |

## 操作指南

### 操作一：查看应用实例列表

**操作步骤**

1. 进入目标工作台，在侧边栏选择目标项目

2. 展开 **边缘应用**，点击 **应用部署**

3. 查看项目中所有已部署的应用实例

**操作结果**

查看到当前项目中所有应用部署的运行状态。

### 操作二：查看应用实例详情

**操作步骤**

1. 在应用实例列表中，点击目标应用名称

2. 进入应用详情页，查看以下信息：
   - 应用基本信息（名称、版本、创建时间）
   - 工作负载运行状态
   - 容器组列表和状态

**操作结果**

了解应用的运行详情和资源使用情况。

### 操作三：删除应用实例

**操作步骤**

1. 在应用实例列表中，找到目标应用

2. 点击操作列的 **删除** 按钮

3. 在确认对话框中，确认删除操作

**操作结果**

应用实例及其关联的工作负载被删除，资源释放。

## 常见问题

### 问题：应用实例状态显示异常
**现象**：应用状态不是「运行中」
**原因**：可能是镜像拉取失败、资源不足或配置错误
**解决**：点击应用名称进入详情页，查看工作负载的事件信息定位具体原因

### 问题：删除应用实例后资源未释放
**现象**：删除应用后项目资源配额未减少
**原因**：应用关联的存储卷可能未被自动清理
**解决**：在项目的存储管理中手动清理不再使用的存储卷
