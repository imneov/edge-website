# 应用部署

> **导航路径**: 应用商店 > 选择应用 > 部署
> **所需权限**: 租户空间管理员或应用管理权限

## 功能说明

应用部署允许您从应用商店选择应用，选择版本并部署到指定的项目中。部署完成后，应用将作为工作负载运行在边缘节点上。

## 操作指南

### 操作一：从应用商店部署应用

**操作步骤**

1. 在应用商店中，点击目标应用进入详情页

2. 点击页面右上角的 **部署** 按钮

3. 在部署配置页面，填写以下信息：

   | 字段 | 是否必填 | 说明 |
   |------|----------|------|
   | 版本 | 是 | 选择要部署的应用版本 |
   | 目标租户空间 | 是 | 选择部署到哪个租户空间 |
   | 目标项目 | 是 | 选择部署到租户空间内的哪个项目 |
   | 应用名称 | 是 | 为本次部署命名 |

4. 根据应用要求，配置应用参数

5. 点击 **确认** 开始部署

**操作结果**

应用部署成功后，您可以在对应项目的「工作负载」页面查看部署状态。

### 操作二：从租户空间应用模板部署

**操作步骤**

1. 进入目标工作台，在侧边栏展开 **边缘应用**，点击 **应用模板**

2. 在应用模板列表中，点击目标模板名称

3. 在模板详情页面，选择版本并点击 **部署**

4. 填写部署配置并确认

**操作结果**

应用部署成功后，在工作台的「工作负载」页面可以查看运行状态。

## 常见问题

### 问题：部署按钮不可点击
**现象**：应用详情页的「部署」按钮为灰色不可点击
**原因**：当前用户没有任何租户空间的部署权限
**解决**：联系管理员为您分配租户空间的应用管理权限

### 问题：部署后应用状态异常
**现象**：部署完成后工作负载状态不是「运行中」
**原因**：可能是镜像拉取失败、资源配额不足或配置错误
**解决**：进入工作负载详情页，查看事件日志中的错误信息
