# 常见问题

## 应用商店问题

### 问题：应用商店页面为空
**现象**：进入应用商店后没有任何应用
**原因**：平台尚未上架任何应用
**解决**：联系平台管理员在应用管理中上架应用

### 问题：搜索应用没有结果
**现象**：输入关键词后搜索结果为空
**原因**：应用名称不匹配或应用未上架
**解决**：检查关键词拼写；联系管理员确认应用是否已上架

## 应用部署问题

### 问题：部署按钮不可点击
**现象**：应用详情页的「部署」按钮为灰色
**原因**：当前用户没有任何租户空间的部署权限
**解决**：联系管理员为您分配租户空间的应用管理权限

### 问题：部署后应用状态异常
**现象**：部署完成后，应用实例状态不是「运行中」
**原因**：可能是镜像拉取失败、资源配额不足或配置参数错误
**解决**：
1. 进入应用实例详情页
2. 查看工作负载的事件信息
3. 根据事件中的错误信息定位问题

### 问题：部署时找不到目标项目
**现象**：部署配置中项目列表为空
**原因**：当前用户在选择的租户空间中没有可用的项目
**解决**：联系租户空间管理员创建项目或分配项目权限

## 版本管理问题

### 问题：版本提交后长时间未审核
**现象**：版本状态一直是「待审核」
**原因**：平台管理员尚未处理审核请求
**解决**：联系平台管理员在应用审核页面处理待审核版本

### 问题：版本被驳回
**现象**：提交的版本状态变为「已驳回」
**原因**：版本内容不符合平台审核标准
**解决**：查看驳回原因，根据反馈修改后重新提交

## 应用模板问题

### 问题：看不到新建应用模板按钮
**现象**：应用模板页面没有新建按钮
**原因**：当前用户没有该租户空间的应用管理权限
**解决**：联系管理员分配租户空间管理权限

### 问题：上传版本失败
**现象**：上传版本包时报错
**原因**：版本包格式不正确或文件大小超过限制
**解决**：确认版本包格式正确；如文件过大，尝试压缩后重新上传
